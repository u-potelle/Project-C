# PROJECT C

Ce projet vise à développer une application en ligne de commande en C, conçue pour parcourir une base de données de super-héros au format JSON. Il s'agit avant tout d'une mise en pratique des notions essentielles du langage C, telles que la gestion de la mémoire, la manipulation des chaînes de caractères, l'utilisation des pointeurs, les structures de contrôle, ainsi que l'intégration de bibliothèques externes.

Technologies et bibliothèques utilisées :
    • Langage C
    • cJSON — bibliothèque légère de parsing JSON en C (DaveGamble/cJSON)
    • readline — saisie interactive en ligne de commande avec historique
    • Compilation via GCC

Le projet intègre l'ensemble des fonctionnalités prévues dans le cahier des charges, des sections II à IV. Il permet notamment d'afficher la liste des héros, de les comparer entre eux, de gérer une sélection de favoris et de proposer un quiz interactif.
